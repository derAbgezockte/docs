## 🎛️ Using the XionHosting Control Panel

Our custom control panel lets you manage all your services in one place.

### 🔧 Main Features
- Start/Stop/Reboot servers
- View resource usage
- Reinstall OS

### 📁 File Management
You can access your files via:
- Web File Manager
- SFTP Access

### 🧠 Troubleshooting
Check logs and monitor CPU/RAM/Network in real time from the dashboard.
